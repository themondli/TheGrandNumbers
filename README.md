# TheGrandNumbers
A Series of Different Math Projects Written in Python
Python Math & Combinatorics Mini-Projects

This repository contains a collection of Python scripts implementing various mathematical and combinatorial problems:

GrandNumbers – Explore properties of large numbers and sequences.
sumOfIndices – Calculate the sum of indices based on certain conditions.
numberOfChickenHeads – Solve the classic chicken head counting problem.
numberOfRabbitHeads – Solve the classic rabbit head counting problem.
Birthday Problem Combinatorics – Compute probabilities related to shared birthdays among a group of people.

Each script is self-contained and demonstrates practical Python implementations of these concepts.

Requirements
Python 3.x

Usage
Run each script individually
